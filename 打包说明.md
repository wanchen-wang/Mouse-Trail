# 轨迹精灵 - 打包说明

## 方法一：使用打包脚本（推荐）

### Windows系统：
直接双击运行 `build.bat` 文件，或在命令行中执行：
```bash
build.bat
```

### Linux/Mac系统：
```bash
chmod +x build.sh
./build.sh
```

## 方法二：手动打包

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 执行打包
```bash
pyinstaller main.spec
```

或者使用完整命令：
```bash
python -m PyInstaller main.spec
```

### 3. 打包结果
打包完成后，可执行文件将位于 `dist/轨迹精灵.exe` (Windows) 或 `dist/轨迹精灵` (Linux/Mac)

## 注意事项

1. **PyInstaller版本**：建议使用 PyInstaller 5.0 或更高版本
2. **Python版本**：需要 Python 3.8 或更高版本
3. **管理员权限**：某些情况下可能需要以管理员权限运行（特别是Windows）
4. **杀毒软件**：打包后的exe可能被某些杀毒软件误报，这是正常现象
5. **文件大小**：由于包含了PyQt6，最终exe文件可能较大（约50-100MB）

## 故障排除

### 如果打包失败：
1. 检查是否所有依赖都已正确安装
2. 尝试清理旧的构建文件：删除 `build` 和 `dist` 文件夹
3. 检查是否有中文路径或特殊字符
4. 查看 `build/main/warn-main.txt` 文件中的警告信息

### 如果exe运行失败：
1. 在命令行中运行exe查看错误信息（如果设置了console=True）
2. 检查是否缺少DLL文件
3. 尝试在打包时添加 `--onefile` 参数（虽然spec文件中已经设置）

## 自定义设置

如需修改打包配置，可以编辑 `main.spec` 文件：
- `name='轨迹精灵'` - 修改exe文件名
- `console=False` - 是否显示控制台窗口
- `icon=None` - 添加应用图标（需要.ico文件）
