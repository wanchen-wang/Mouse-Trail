# GitHub发布说明 - Mac用户必读

## ⚠️ 重要：Mac用户请先看这里

如果你在Mac上下载并运行应用时遇到"mac不支持此应用程序"的错误，
**这是正常的**，请按照以下步骤解决：

## 🚀 快速解决方案（3选1）

### 方法1：右键打开（最简单）⭐⭐⭐
1. 找到 `轨迹精灵.app`
2. **右键点击**（不要直接双击）
3. 选择"打开"
4. 在警告对话框中点击"打开"

### 方法2：终端命令（推荐）⭐⭐
打开终端，执行：
```bash
xattr -cr ~/Downloads/轨迹精灵.app
```
（如果应用在其他位置，请修改路径）

### 方法3：系统设置
系统设置 > 隐私与安全性 > 允许运行

## 📋 详细文档

- **快速开始**：查看 `README_MAC.md`
- **详细指南**：查看 `Mac运行指南.md`
- **给朋友看**：查看 `给Mac用户的一封信.txt`

## 🔧 为什么需要这样做？

macOS的Gatekeeper安全机制会隔离未签名的应用。
这是正常的安全行为，不是bug。

## 📦 打包Mac版本

如果你要在Mac上重新打包：

```bash
# 1. 安装依赖
pip3 install -r requirements.txt

# 2. 使用Mac专用脚本打包
chmod +x build_mac.sh
./build_mac.sh

# 打包完成后会自动移除隔离属性
```
