# Mac用户快速开始指南

## 🚀 快速运行（3步）

### 步骤1：下载应用
从GitHub下载 `轨迹精灵.app` 文件

### 步骤2：移除安全限制
打开**终端**，执行：
```bash
xattr -cr ~/Downloads/轨迹精灵.app
```
（如果应用在其他位置，请修改路径）

### 步骤3：运行应用
- **方法A**：直接双击应用
- **方法B**：如果双击不行，右键点击 → 选择"打开"

## ⚠️ 如果遇到"无法打开"错误

### 最简单的方法：
1. 右键点击 `轨迹精灵.app`
2. 选择"打开"
3. 在警告对话框中点击"打开"

### 或者使用终端：
```bash
# 进入应用所在目录
cd ~/Downloads  # 或应用所在的其他目录

# 移除隔离属性
xattr -cr 轨迹精灵.app

# 现在可以运行了
open 轨迹精灵.app
```

## 📋 系统要求

- macOS 10.13 或更高版本
- 需要授予"辅助功能"权限（用于全局快捷键）

## ⌨️ 快捷键

- **⌘+O**: 显示/隐藏控制面板
- **⌘+P**: 退出程序

## 🔧 权限设置

首次运行后，前往：
**系统设置 > 隐私与安全性 > 辅助功能**

勾选应用以启用全局快捷键功能。

## ❓ 常见问题

**Q: 为什么需要执行xattr命令？**
A: macOS的安全机制会隔离未签名的应用，这个命令可以移除隔离属性。

**Q: 执行命令后还是无法运行？**
A: 尝试右键点击应用 → 选择"打开"，然后在警告中点击"打开"。

**Q: 快捷键不工作？**
A: 请确保已在"系统设置 > 隐私与安全性 > 辅助功能"中授予权限。
