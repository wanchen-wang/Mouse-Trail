# 鼠标拖尾特效程序 (Mouse Trail)

本项目使用 PyQt6 实现，具有高性能渲染和点击穿透功能。
!仅限windos系统使用

### 🚀 快速开始
* **[点击下载 exe 执行文件][https://github.com/wanchen-wang/Mouse-Trail/releases/tag/v1.2]**
* **退出方式**：按下键盘 `Ctrl + P`
